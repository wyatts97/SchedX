# SchedX Storybook

Welcome to the SchedX component library documentation!

This Storybook instance contains all the reusable components used in the SchedX application.

## Getting Started

Browse the components in the sidebar to see:
- **Component documentation** - Learn how each component works
- **Interactive examples** - Play with component props in real-time
- **Code snippets** - Copy-paste ready code examples

## Running Storybook

To run Storybook locally:

```bash
npm run storybook:dev
```

To build a static version:

```bash
npm run storybook:build
```

## Component Structure

Components are organized by their location in the project:
- **Components/** - Reusable UI components from `src/lib/components`
- **Pages/** - Page-level components

## Key Components

### ImageEditor
A full-featured image editor modal using TUI Image Editor with support for:
- Cropping, rotating, and flipping
- Drawing and shapes
- Text and icons
- Filters
- Portrait and landscape images

Check out the ImageEditor stories to see it in action!
